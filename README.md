<h1>Active Directory HomeLab</h1>

<h2>Description</h2>
Creating a active directory homelab to better understand active directory and windows networking.
<br />


<h2>Utilities Used</h2>

- <b>Oracle VirtualBox</b> 
- <b>Windows 10 ISO</b>
- <b>2019 Server ISO</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
